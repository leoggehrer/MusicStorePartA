# MusicStore
Das Beispiel MusicStore ist ein Fallbeispiel für die Erstellung eines Software-Systems mit C#. 
Die Anwendung besteht aus unterschiedlichsten Entitäten 

+ Teil A bildet die Basis für die weiteren Entwicklungsschritte und den Ausbaustufen 
+ Teil B ist die Erweiterung von Teil A mit asynchronen Methoden.
+ Teil C erweitert die Architektur um eine Web-Api
+ Teil D integriert einen Adapter, welcher den Zugriff auf die Geschäftslogik direkt oder über das Web-Api abstrahiert.
+ Teil E fügt eine Mvc-Client zur Anwendung hinzu
+ Teil F verwendet einen Blazor-Client als Präsentationsschicht
+ Teil G zeigt wie die gesamte Anwendungsstruktur durch den Einsatz eines Code-Generator unterstützt werden kann
